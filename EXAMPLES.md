## ğŸ”„ Workflow Tips

### Iterative Refinement

1. **Start broad:**
   ```bash
   ai-advisory ask "What are the best practices for API design?"
   ```

2. **Get specific based on results:**
   ```bash
   ai-advisory ask "How should I version my REST API: URL path, header, or query parameter?"
   ```

### Selective Evaluation

- Review Level 1 responses carefully
- Select 2-3 diverse perspectives for Level 2
- This gives focused consensus while reducing API costs

### Question Crafting

**Good questions:**
- Specific and focused
- Include context when relevant
- Ask for comparisons or recommendations

**Example:**
```bash
ai-advisory ask "For a startup with a team of 3 developers building a mobile app, should we use React Native, Flutter, or native development?"
```

**Less effective:**
```bash
ai-advisory ask "What's the best mobile framework?"
```

---

## ğŸ’¡ Pro Tips

1. **Include constraints**: "with a budget of $10k/month" or "for a team of 2"
2. **Ask for trade-offs**: "pros and cons" or "when to use each"
3. **Request examples**: "with real-world examples"
4. **Specify audience**: "explain to a beginner" or "for an expert"
5. **Set context**: "for a healthcare app" or "in a startup environment"
